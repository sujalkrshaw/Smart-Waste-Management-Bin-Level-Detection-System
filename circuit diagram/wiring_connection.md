# ESP32 Wiring Connections

## HC-SR04 Ultrasonic Sensor

| HC-SR04 | ESP32   |
| ------- | ------- |
| VCC     | 5V      |
| GND     | GND     |
| TRIG    | GPIO 5  |
| ECHO    | GPIO 18 |

## LEDs

| LED        | ESP32   |
| ---------- | ------- |
| Green LED  | GPIO 26 |
| Yellow LED | GPIO 27 |
| Red LED    | GPIO 14 |

## Buzzer

| Buzzer   | ESP32   |
| -------- | ------- |
| Positive | GPIO 25 |
| Negative | GND     |
